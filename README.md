
# Jogo do Mario com IA

Este projeto é uma recriação interativa do jogo do Mario, desenvolvida com foco educativo para demonstrar a aplicação de Inteligência Artificial (IA) no controle de jogos por meio de movimentos corporais.

Através do uso do PoseNet (modelo de detecção de poses humanas da biblioteca ml5.js) e da P5.js, o jogo interpreta a posição do nariz do jogador como um controle de movimento — permitindo andar e pular sem a necessidade de teclado ou joystick.

A proposta é explorar de forma lúdica como redes neurais podem ser integradas a aplicações visuais para criar interfaces alternativas, inclusivas e criativas, aproximando o aprendizado de Machine Learning do desenvolvimento de jogos.

-Além do aprendizado técnico, o projeto também reforça conceitos de:

-Visão computacional aplicada à interação humano-computador;

-Processamento de vídeo em tempo real com P5.js;

-Uso de bibliotecas de IA acessíveis no navegador;

-Experimentos em design de interação baseado em movimento.


## Stack utilizada

**Front-end:** html, css, javascript, P5.js e seus complementos (p5.dom.js ou p5.sound.js). 
Ajax e Bootstraps também foram usadas.
